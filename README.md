# 🍽️ Swiggy Sales Analysis 📊

## 📌 Overview
This project analyzes Swiggy sales data using Python to uncover patterns in customer orders, revenue trends, and weekly performance.

## 🎯 Objectives
- Analyze daily and weekly sales trends
- Identify peak order periods
- Understand revenue fluctuations
- Generate actionable business insights

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Key Analysis
- 📅 Daily Sales Trend
- 📈 Weekly Trend Analysis
- 💰 Revenue Insights

## 🔍 Key Insights
- Peak orders observed on weekends
- Certain weeks show higher revenue spikes
- Sales trends indicate demand fluctuations
- Useful for inventory and staffing planning
